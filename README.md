# DAIMS
DAIMS Project web resource and public workspace

## Exploring Ethics and Moral Perspectives on AI in Cultural Collections - workshop

This repository contains the slides from the [workshop held at the University of Leeds 12-06-2026](https://leeds.libcal.com/event/4544345), in both Powerpoint (with accessible descriptions) and PDF (note that the Powerpoint is a more accessible format, but the PDF may be more suitable for printing).

**Please note:** Any attendees poll responses have been removed, but the slides still contain references to a now-closed Vevox session etc. These slides are meant purely as a record for attendees: we will shortly be uploading tidied/reorganised versions that are more useful as an open educational resource, with appropriate licensing etc.
